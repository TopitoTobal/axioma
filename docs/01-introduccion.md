# Introduccion

**Axioma** es un lenguaje de programacion interpretado, de codigo abierto, disenado para que hispanohablantes puedan programar en su idioma nativo.

## Filosofia

La mayoria de los lenguajes de programacion usan palabras clave en ingles (`if`, `while`, `for`, `function`, `class`). Axioma rompe esa barrera permitiendo escribir codigo completamente en espanol.

## Caracteristicas Principales

- **Keywords en espanol** — Todas las palabras reservadas del lenguaje estan en espanol
- **Interpretado** — No necesita compilacion, se ejecuta directamente
- **Tipado dinamico** — Las variables pueden contener cualquier tipo de dato
- **Orientado a objetos** — Soporta clases, herencia y metodos
- **Recursivo** — Las funciones pueden llamarse a si mismas
- **Codigo abierto** — Licencia MIT, libre de usar y modificar

## Palabras Clave

| Palabra | Significado | Equivalente en ingles |
|---------|-------------|----------------------|
| `var` | Declarar variable | `let` / `var` |
| `funcion` | Definir funcion | `function` |
| `retornar` | Devolver valor | `return` |
| `si` | Condicional | `if` |
| `sino` | Sino/otherwise | `else` |
| `mientras` | Bucle condicional | `while` |
| `para` | Bucle con contador | `for` |
| `imprimir` | Mostrar en pantalla | `print` |
| `verdadero` | Booleano true | `true` |
| `falso` | Booleano false | `false` |
| `nulo` | Valor nulo | `null` |
| `y` | AND logico | `and` / `&&` |
| `o` | OR logico | `or` / `\|\|` |
| `no` | NOT logico | `not` / `!` |
| `clase` | Definir clase | `class` |
| `este` | Referencia a instancia | `this` |
| `nuevo` | Crear instancia | `new` |
| `heredar` | Herencia de clase | `extends` |
| `salir()` | Salir del REPL | `exit()` |

## Influencias

Axioma toma lo mejor de:

- **Python** — Sintaxis limpia y tipado dinamico
- **JavaScript** — Flexibilidad y paradigma de objetos
- **Lua** — Simplicidad y facilidad de embedding
